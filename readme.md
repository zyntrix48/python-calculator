# Калькулятор

Простой консольный калькулятор на Python, который выполняет основные арифметические операции.

## Функциональность

- ➕ Сложение
- ➖ Вычитание  
- ✖️ Умножение
- ➗ Деление

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы:
```bash
git clone https://github.com/zyntrix48/python-calculator
```

2. Убедитесь, что у вас установлен Python 3.6+

3. Запустите программу:
```bash
python main.py
```

## Использование

1. Программа запросит два числа
2. Выберите операцию из списка:
   - 1 + (сложение)
   - 2 - (вычитание)
   - 3 * (умножение)
   - 4 / (деление)
3. Введите номер операции
4. Получите результат

### Пример работы:
```
Enter first number: 10
Enter second number: 5
1. +
2. -
3. *
4. /
Enter operator(enter index): 3
50.0
```

- `sub(a, b)` - вычитание
